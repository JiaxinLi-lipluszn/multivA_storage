C5 · 解码器遮蔽保持自回归

*C5: Decoder mask 不是装饰,它是在守住自回归边界。*

Transformer decoder 虽然不用 RNN,但仍然要像语言模型一样预测下一个 token:当前位置不能偷看未来答案。

关键直觉: self-attention 本来是“全员互看”。

在 encoder 里这很好:每个输入位置都能看整句。

但在 decoder 里,如果位置 `i` 能看 `i+1, i+2...`,训练时就会把未来真实输出泄漏进来,自回归预测就不成立了。

Scaled dot-product attention 里可以在 softmax 前加 mask,把非法连接直接排除。

论文的机制很具体:不是“提醒模型别看”,而是把未来位置的 attention logit 设成 `-∞`。

> We implement this inside of scaled dot-product attention by masking out (setting to −∞) all values in the input of the softmax which correspond to illegal connections.

softmax 后这些位置权重就是 0,信息流被切断。

再加一个关键细节:decoder 的输出 embedding 是错位输入的。

> This masking, combined with fact that the output embeddings are offset by one position, ensures that the predictions for position i can depend only on the known outputs at positions less than i.

所以预测第 `i` 个 token 时,可用的是已知的 `y_

常见误解:mask 不是让 decoder 在生成时“一次吐完整句”。

它保证的是*训练时并行计算每个位置的表示,但每个位置的信息边界仍像自回归模型一样合法*。

推理时仍然按 token 逐步生成,因为下一个输入 token 要等上一步采样/选择出来。

收尾一句:Transformer decoder 的巧妙处在这里。

它用矩阵化 self-attention 拿到训练并行性,又用 causal mask 保住因果方向:

`prediction at i` ← only `known outputs < i`。
